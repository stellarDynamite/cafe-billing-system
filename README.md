# ☕ Café Billing System

An interactive console-based café billing system built in Java.  
It allows users to order multiple items (Cake & Coffee), customize selections, and calculates the total bill based on user input.

---

## 🛠️ Built With
- Java OOPs

---

## 🚀 Features
- Interactive console-based menu  
- Multiple item ordering  
- Basic customization (e.g., coffee add-ons, cake options)  
- Real-time price calculation  
- Loop-based ordering system  
- Final bill generation  

---

## 📁 Project Structure
.
├── src/
│ ├── Cafe.java # Main entry point (runs the program)
│ ├── Order.java # Handles menu, user input, and billing logic
│ ├── Item.java # Base class for items (price handling)
│ ├── Cake.java # Cake-related logic and pricing
│ └── Coffee.java # Coffee-related logic and customization
└── README.md

---

## ⚙️ Getting Started

### Prerequisites
- Java installed (JDK 8 or above)

### Installation & Run

```bash
git clone https://github.com/stellarDynamite/cafe-billing-system.git
cd cafe-billing-system/src
javac *.java
java Cafe
```
